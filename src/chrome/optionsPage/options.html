This extension is focused around the 'heavy' subscription user, the majority of these tweaks change how you interact with your subscriptions videos. If you would like automatic quality select, full buffering and many more features focused more around watching videos I suggest you look at the Youtube Center extension.

This adds a 'remove' button next to each video from your subscriptions. When clicked the video is removed using youtubes 'Hide video' option. This means that it will remain hidden if viewed from a browser without this extention installed.

Enable the 'Remove' button next to each video.

Change the YouTube icon URL - This changes the page that clicking the YouTube icon (top left of all youtube pages) takes you to. Usualy this will take you to "youtube.com" which shows you the "whats new" page. Enabling this will allow you to change the page that this icon takes you, to the one specified in the textbox below.

Redirect "youtube.com" - If your browser takes you to "YouTube.com" then you will always be redirected to the URL below.

This allows the user to remove any previously watched videos from there home-feed in one of two ways. When in non-automatic mode, it adds a button to the left side of the home page which says 'Remove watched videos'. When clicked it will hide all watched videos from you subscriptions.In automatic mode, all watched videos will be removed automatically when you load the YouTube homepage.

This loads all of your subscription videos into view on your homepage, allowing you to view them all in one continuous feed. This can be used to browse through your rescent subscriptions to remove any you dont want or find those you may have missed.

FAQ's and Client side fixes.

Settings not saving

This can be caused by a sync error between local and cloud storage for the settings of the extension. For a fix for this please look here.

Please read when reporting errors/bugs.

When reporting errors with YouTweak it is very helpful to me when fixing them if a list of any errors was included with the description of the problem.

To do this press Ctrl + Shift + J to open the console, here any errors will be listed. A copy-paste of this infomation will help identify the problem. I would suggest using the site PasteBin and to include the share link in the problems description. To all those who do submit error reports thank you for helping to make this extension better.
