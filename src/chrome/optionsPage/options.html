There is no way to re-add removed videos to the "My Subscriptions" page, to find the video again you would have to search the YouTubers channel.

FAQ's and Bug's

About this extension.

This extension is focused around the 'heavy' subscription user, the majority of these tweaks change how you interact with your subscriptions videos. If you would like automatic quality select, full buffering and many more features focused more around watching videos I suggest you look at the Youtube Center extension.


Hide the extension icon.

Follow this guide to learn how to remove the icon from the top of chrome (Options menu still acessable from the extensions tab).

Settings not saving

This can be caused by a sync error between local and cloud storage for the settings of the extension. For a fix for this please look here.

Please read when reporting errors/bugs.

Firstly please check that using the previous version of the extension (by the 'Version Selection' combobox does not fix your problem.

If you wish to contact me with any issues or suggestions then please leave me a message on the Github Issues page as this will be the place that I am most likely to see it.

Leaving a problem on the chrome webstore is not the best solution as I do not recieve any notification when a new problem is listed and it may be a while before I see it.
