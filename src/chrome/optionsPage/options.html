YouTweak The subscription manager

Hi, thanks for downloading YouTweak! This extension aims to provide an improved experience for the 'heavy' subscription user, the majority of these tweaks change how you interact with your subscriptions page. If you would like automatic quality select, full buffering and many more features focused more around watching videos I suggest you look at one of the many extensions which already accomplish this.

Development on this extension is ongoing but sporadic as I have had a full time job for the past year & am now returning to uni. The source code is available on GitHub and I welcome any bug fixes or feature suggestions you might have.

Extension Settings
This page saves automatically on exit!

Remove individual subscription video

This feature makes it easier to delete unwanted videos from your YouTube homepage. The current YouTube layout seems to discourage the use of the 'Hide video' option previous layouts featured prominently. This feature intends to fix this, giving the user a per-video dedicated button for the removal of the video. This button instigates YouTube's own hide video feature which means that hiding it with this app also hides it if you were to sign in from a different computer without the app, however this does mean there is no way to recover a deleted video to your subscriptions page.

Enabling this feature will add an 'X' into the bottom left of each video thumbnail on your subscriptions page. Clicking this button will remove the video.

Potential issues notice! #1

Please understand that this extension relies on YouTube's ability to hide videos from their homepage. From time to time, YouTube seem to 'mess up' this feature which causes videos to not be removed when the user tries to delete them. If that is the case then this feature of the extension may not function properly as YouTweak relies on this functionality. If this is the case then there is nothing which I can do to fix the extension to work & the only solution is to wait for YouTube to fix it their end.

Potential issues notice! #2

The functionality of removing subscription videos is currently compromised by an ongoing (*read intentional*) issue with YouTube which has been present for many since 2014. While the button to remove videos will still be present, videos older than a week may 'pop back' on page refresh & refuse to be deleted. As this is a problem with YouTubes delete mechanisms there is little I can do to fix this & I suggest using the Collapsible Subscription Groups feature to automatically collapse videos which cant be hidden. Please see this issue for more details.
Remove watched videos

This feature makes your homepage much cleaner by removing any subscription videos that you have already watched. When activated you will notice a new button added to the left navigation column of the homepage. When clicked any previously watched videos in your current subscriptions are removed. This button instigates YouTube's own hide video feature which means that hiding it with this app also hides it if you were to sign in from a different computer without the app, however this does mean there is no way to recover a deleted video to your subscriptions page.

Enabling the automatic video removal feature will remove the button from the left menu & instead continually check for any newly watched videos & remove them as soon as they are found.

Please note! You need to have "Watch History" unpaused for this feature to work as it uses YouTube's data to see if a video has been watched or not. This can be enabled in your History tab.

Load all subscription videos

This adds a button to the homepage which when clicked loads all of the pages of your subscriptions into view, allowing you to browse them all without having to reload the page. When clicked this will load all available subscription videos onto the page for you to browse in one continuous feed.

Remove all subscription videos

This adds a button to the homepage which when clicked will cycle through all of your subscriptions and remove them, leaving you with a clean homepage ready for new subscriptions to be easily visible. (Example for use could be after returning from holiday and clearing the weeks of subscription buildup, or when a channel uploads many videos that clog-up your subscription feed)

Potential issues notice! #1

While the button to remove videos will still be present, videos older than a week may 'pop back' on page refresh & refuse to be deleted.

Therefore, this feature will currently only remove all videos within the last week. I suggest using the Collapsible Subscription Groups feature to automatically collapse videos which cant be hidden. Please see this issue for more details.
YouTube Subscriptions as default home page

This changes the URL that the YouTube icon at the top left of every YouTube page links to. The default link is to "youtube.com" which shows all of your subscriptions and recommended videos mixed together. By default the app redirects to "youtube.com/feed/subscriptions" which only shows your subscriptions videos (but this can be set manually to any URL).


Collapsible Subscription Groups

This allows you to collapse the 'Today', 'Yesterday', etc subscription groups which are present on the grid subscription view. A collapse button will be added to the left of the group name which will toggle the visible state of the groups videos. You can also elect to have videos older than 'This Week' automatically collapsed when the page loads.


Updates

Due to the constant changes implemented by YouTube it will be necessary to update this extension to reflect these changes. The code for this extension is open source and will be commited to my GitHub repo whenever there is an update or a significant change has been made.
